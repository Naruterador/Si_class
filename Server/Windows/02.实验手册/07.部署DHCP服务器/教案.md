# 《DHCP服务器的部署与地址分配》教案  
（以 Windows Server 2016 为教学案例）

---

### 一、课程内容
认识 DHCP（动态主机配置协议）的基本原理与作用，掌握在 Windows Server 2016 中安装与配置 DHCP 服务角色的全过程，能独立创建作用域、配置地址池、排除范围与保留地址，并验证客户端自动获取 IP 的效果。

---

### 二、课时安排
共 2 课时（理论讲授 + 实验操作）

---

### 三、教学方式
讲授 + 案例演示 + 任务驱动 + 小组操作  

---

### 四、教学目标
#### （一）知识目标
- 理解 DHCP 的工作机制与核心作用；  
- 掌握 DHCP 服务器角色的安装方法；  
- 掌握配置作用域、地址池、排除范围与保留地址的流程；  
- 理解 DHCP 授权、租约与地址冲突的概念。  

#### （二）能力目标
- 能够独立安装与配置 DHCP 服务器；  
- 能根据网络规划创建合理的地址分配方案；  
- 能使用命令或图形界面验证客户端自动分配 IP 成功；  
- 能分析并排查 DHCP 地址分配失败的常见原因。  

#### （三）素质目标（含思政元素）
- 通过 DHCP 自动化配置的学习，培养学生科学规划与规范管理网络资源的意识；  
- 树立“以服务为核心、以效率为导向”的职业理念；  
- 理解智能分配与集中管理对提升信息化水平的重要意义。  

---

### 五、教学重点
- DHCP 作用域、地址池与排除范围的配置；  
- DHCP 地址租约与保留地址机制；  
- 客户端自动获取 IP 的验证方法。  

---

### 六、教学难点
- 理解 DHCP 授权机制及其与 AD 域的关系；  
- 掌握 DHCP 地址分配的优先级与排除规则。  

---

### 七、教学方法
讲授法、演示法、任务驱动法、实验验证法  

---

### 八、教学过程

#### （一）导入环节（约10分钟）
1. 教师提问：“在企业网络中，如果每台电脑都要手动设置 IP，会有什么问题？”  
2. 学生回答后总结引出：IP 冲突、配置效率低、管理困难。  
3. 教师说明：DHCP 可自动分配 IP，提高效率与安全性，是网络管理的重要基础服务。  

---

#### （二）讲解示范环节（约50分钟）

##### 1. 实验环境与拓扑说明
- 实验拓扑：

| 机器名 | 系统版本 | IP | 作用 |
|---------|-----------|-----------|--------|
| iLgj-Server-xx-01 | Win2016 | 192.168.xx.10 | DHCP 服务器 |
| iLgj-Server-xx-02 | Win2016 | 自动获取 | DHCP 客户端1 |
| iLgj-Server-xx-03 | Win2016 | 自动获取 | DHCP 客户端2 |

- 所有主机同网段，网关与 DNS 可选配置为 192.168.xx.1 / 192.168.xx.10。  

---

##### 2. DHCP 服务角色安装与配置
1. 在 iLgj-Server-xx-01 上安装 **DHCP Server** 角色。  
   - 打开“服务器管理器” → 添加角色 → 勾选 DHCP Server。  
   - 安装完成后截图。  
2. 打开 **DHCP 管理器** → 右键 IPv4 → 新建作用域。  
   - 名称：Scopexx；  
   - 地址池：起始 192.168.xx.100 ～ 192.168.xx.200；  
   - 子网掩码：255.255.255.0；  
   - 默认网关：192.168.xx.1；  
   - DNS 服务器：192.168.xx.10。  
   - 完成后截图作用域属性页面。  
3. 添加 **排除范围**：192.168.xx.100 ～ 192.168.xx.110。  
   - 作用：防止与静态地址冲突。  
4. 完成 DHCP **授权**（右键服务器 → 授权）。  
   - 授权状态变为“绿色箭头”表示启用成功。  

---

##### 3. DHCP 地址保留配置
1. 在作用域中选择“保留” → 新建保留 → 名称为 `Client03`；  
2. 指定 MAC 地址并分配固定 IP：192.168.xx.112；  
3. 保留设置完成后截图结果。  

---
