# 《认识 Active Directory 与域环境》教案  
（以 Windows Server 2016 为教学案例）

---

### 一、课程内容
理解工作组与域的区别，掌握 Active Directory 域服务（AD DS）的作用与结构，能够完成域控制器的安装与提升操作，学会在域环境中创建组织单位（OU）与用户账户，并让客户端计算机加入域，体验集中管理机制。

---

### 二、课时安排
共 2 课时（理论讲授 + 实验操作）

---

### 三、教学方式
讲授 + 演示 + 任务驱动 + 小组实验  

---

### 四、教学目标
#### （一）知识目标
- 理解“工作组”和“域”的区别与应用场景；  
- 掌握 Active Directory 域服务（AD DS）的作用、组成结构及功能；  
- 理解域控制器（DC）、组织单位（OU）、域用户账户等核心概念；  
- 掌握服务器提升为域控制器与客户端加入域的操作流程。  

#### （二）能力目标
- 能够独立安装与配置 AD DS 角色；  
- 能将服务器提升为域控制器；  
- 能创建 OU 与域用户账户；  
- 能让客户端加入域并验证域登录功能。  

#### （三）素质目标（含思政元素）
- 通过域集中管理机制的学习，培养学生“统一规划、分级管理”的系统思维；  
- 理解组织架构管理在信息化建设中的价值，增强团队协作意识；  
- 树立安全管理与责任担当理念，认识到“规范即安全、集中即高效”的职业精神。  

---

### 五、教学重点
- 域控制器的安装与配置；  
- 客户端计算机加入域的过程；  
- 组织单位与用户账户的创建与管理。  

---

### 六、教学难点
- 理解 AD DS 的层级结构与对象组织方式；  
- 理解域用户登录验证的机制与安全策略。  

---

### 七、教学方法
讲授法、演示法、任务驱动法、实验验证法  

---

### 八、教学过程

#### （一）导入环节（约10分钟）
1. 教师提问：“如果企业中有上百台电脑，每次都要单独设置用户与密码，会不会很麻烦？”  
2. 引导学生回顾工作组环境的局限性：用户分散、权限重复设置、管理效率低。  
3. 教师总结：Active Directory 域服务（AD DS）通过集中认证与授权解决了这些问题。  

---

#### （二）讲解示范环节（约50分钟）

##### 1. 实验拓扑与环境
在 VMware Workstation 中部署三台虚拟机：

| 机器名 | 系统版本 | 内存 | 硬盘 | CPU | 角色 |
|---------|-----------|------|------|------|------|
| iLgj-Server-xx-01 | Win2016 | 2G | 50G | 1核 | 域控制器（DC） |
| iLgj-Server-xx-02 | Win2016 | 2G | 50G | 1核 | 域成员客户端 |
| iLgj-Server-xx-03 | Win2016 | 2G | 50G | 1核 | 域成员客户端 |

网络配置示例：

| 机器名 | IP | 子网掩码 | 网关 | DNS |
|---------|----|-----------|------|------|
| iLgj-Server-xx-01 | 192.168.xx.10 | 255.255.255.0 | — | — |
| iLgj-Server-xx-02 | 192.168.xx.11 | 255.255.255.0 | — | — |
| iLgj-Server-xx-03 | 192.168.xx.12 | 255.255.255.0 | — | — |

---

##### 2. 安装 Active Directory 域服务（AD DS）
1. 在 iLgj-Server-xx-01 上安装 AD DS 角色：  
   - 打开“服务器管理器” → 添加角色 → 选择“Active Directory 域服务” → 安装。  
2. 安装完成后点击“将此服务器提升为域控制器”。  
   - 新建林：`contosoxx.com`；  
   - 功能级别：Windows Server 2016；  
   - 设置目录服务恢复密码；  
   - 完成后服务器自动重启。  
3. 登录后打开“Active Directory 用户和计算机”控制台，验证域结构并截图。  

---

##### 3. 客户端加入域
1. 在 iLgj-Server-xx-02 上，打开系统属性 → 计算机名 → 更改 → 选择“域” → 输入 `contosoxx.com`；  
2. 输入域管理员凭据；  
3. 加入成功后系统提示重启，重启后在登录界面选择“contosoxx\Administrator”；  
4. 截图显示“计算机已成功加入域”。  

---

##### 4. 创建组织单位（OU）与域用户
1. 在 iLgj-Server-xx-01 上打开“Active Directory 用户和计算机”；  
2. 在左侧树状结构中右键域名 `contosoxx.com` → 新建 → 组织单位（OU） → 命名为 `ITxx`；  
3. 右键 ITxx → 新建用户：  
   - 用户名：`userxx1`、`userxx2`；  
   - 初始密码：`P@ssw0rd!`；  
   - 取消“用户下次登录时必须更改密码”；  
   - 完成创建并截图。  

---
