### 教学过程
- 教师组织教学、讲授、操作、示范, 以及学生技能学习、活动与操作内容

#### 一、课程导入（XX分钟）
- 上节课我们学习了Linux的文件系统架构，已经对Linux的文件系统有了一个初步的了解。今天我们就来学习一些基本的目录管理命令来帮助我们更好的管理目录。
#### 二、讲解示范（XX分钟）
- 目录管理命令的作用
    - 为了方便管理文件和目录，Linux 系统将它们组织成一个以根目录 / 开始的倒置的树状结构。Linux 中的目录，和 Windows 系统中的文件夹类似，不同之处在于，Linux 系统中的目录也被当做文件看待。
- 目录管理命令介绍
    - pwd
    - mkdir
    - rmdir
    - cd
    - ls
    - tree
    - tar
    - zip/unzip
    - gzip
#### 三、任务实施（XX分钟）
- 根据教师对指令的讲解，指导学生完成实验手册

#### 四、分析总结（XX分钟）
- 本知识点只要介绍了一些常用的系统的命令，让学生对于Linux的操作有了更进一步的了解。
