###  文件和目录类命令
- xx 都为学号


- 利用touch命令，在当前目录创建一个新的空文件newfile_xx(xx为学号)。
`touch newfile_xx`
- pwd命令查看当前所在的目录。
`pwd`

- 用ls命令列出此目录下的文件和目录。
`ls`

- 用 -a 选项列出此目录下包括隐藏文件在内的所有文件和目录。
`ls -a`

- 用man命令查看ls命令的使用手册。
`man ls`

- 在当前目录下，创建测试目录/test_xx。
`mkdir /test_xx`

- 利用ls命令列出文件和目录，确认/test_xx目录创建成功。
`ls /test_xx`

- 进入test_xx目录，利用pwd命令查看当前工作目录。
```shell
cd test_xx
pwd
```


- 利用cp命令复制系统文件/etc/profile到当前目录下。
`cp /etc/profile .`

- 复制文件profile到一个新文件profile_xx.bak，作为备份。
`cp profile profile_xx.bak`

- 用ll命令以长格式列出当前目录下的所有文件，注意比较每个文件的长度和创建时间的不同。
`ll`

- 用less命令分屏查看文件profile_xx.bak的内容。
`less profile_xx.bak`

- 退回上级目录，用tar命令把目录test_xx打包，文件命名为test_xx.tar。
`tar -cvf test_xx.tar test_xx`

- 以gzip格式压缩test_xx.tar，文件命名为test_xx.tar.gz。
`gzip -c test_xx.tar > test_xx.tar.gz`

- 把文件test_xx.tar.gz改名为backup_xx.tar.gz。
`mv test_xx.tar.gz backup_xx.tar.gz`

- 显示当前目录下的文件和目录列表，确认重命名成功。
`ls`

- 把文件backup_xx.tar.gz移动到test_xx目录下。
`mv backup_xx.tar.gz test_xx`

- 显示当前目录下的文件和目录列表，确认移动成功。
`ls`

- 进入test_xx目录，显示目录中的文件列表。
```shell
cd test
ls
```


- 把文件backup_xx.tar.gz解包，显示当前目录下的文件和目录列表。
```shell
tar -xvf backup_xx.tar.gz
ls
```


- 返回上一级目录，复制test_xx目录为testbak_xx目录作为备份。
```shell
cd ..
cp -r test_xx testbak_xx
```


- 查找root用户自己的主目录下的所有名为newfile_xx的文件。
`find /root -name newfile_xx`

- 删除test_xx子目录下的所有文件。
`rm -rf test_xx/*`

- 利用rmdir命令删除空子目录test_xx。
`rmdir test_xx`

- 回到上层目录，利用rm命令删除目录testbak_xx和其下所有文件。
```shell
cd ..
rm -rf testbak_xx
```

### find命令的使用:

- 在/var/lib目录下查找其所有者是games用户的所有文件。
`find /var/lib -user games`

- 在/var目录下查找其所有者是root用户的所有文件。
`find /var -user root`
 
- 查找/usr/bin目录下所有大小超过1 000 000 B的文件并用长格式显示。
`find /usr/bin -size +1000000c -ls`

- 查找/tmp目录下属于root的所有普通文件，这些文件的修改时间为120 min以前，查询结果用长格式显示。 
`find /tmp -type f -user root -mmin +120 -exec ls -l -- {} +`

### 进程管理类命令:

- 显示本用户的进程。
`ps`

- 显示所有用户的进程。
`ps -au`

- 在后台运行cat命令。
`cat &`

- 查看进程cat。
`ps -aux | grep cat`

- 杀死进程cat。
`kill -9 进程号`
- 在 Linux 中，`kill -9` 是一个命令，用于强制终止（杀死）一个进程。这里的 `-9` 是一个信号编号，表示向进程发送 `SIGKILL` 信号。
- 使用 `kill -9` 的情况
- 无法终止进程：当一个进程不能响应标准的 `SIGTERM` 信号（信号编号为 `-15`，它是一个优雅终止信号，允许进程清理资源并正常退出）时，可以使用 `kill -9` 强制终止该进程。
- 僵尸进程：处理无法被正常终止或成为僵尸状态的进程。

- 再次查看进程cat，看其是否已被杀死。
`ps -aux | grep cat`
### 进程管理类命令

- 使用top命令查看和控制进程：
  `top`
  - 用top命令动态显示当前的进程。
  - 只显示用户root的进程（利用u键）。
  - 利用k键，杀死指定top进程。

- 挂起和恢复进程：
  - 执行命令cat。
    `cat`
  - 按Ctrl+z组合键，挂起进程cat。
  - 输入jobs命令，查看作业。
    `jobs`
  - 输入bg，把cat切换到后台执行。
    `bg`
  - 输入fg，把cat切换到前台执行。
    `fg`
  - 按Ctrl+c组合键，结束进程cat。
