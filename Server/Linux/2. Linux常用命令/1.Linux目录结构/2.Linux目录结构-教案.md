#### 一、课程导入（XX分钟）
- Linux文件系统的构造和Windows完全不同，Linux采用“文件夹”的管理模式来规划文件系统，今天就让我们来一起学习一下。

#### 二、讲解示范（XX分钟）

- Linux文件系统目录结构
    - Linux系统中一切都是文件。
    - 在Linux系统中并不存在C/D/E/F等盘符，Linux系统中的一切文件都是从“根（/）”目录开始的。
    - Linux系统中的文件和目录名称是严格区分大小写的。例如，root、rOOt、Root、rooT均代表不同的目录，并且文件名称中不得包含斜杠（/）。
    - Linux系统文件没有扩展名。
- Linux系统文件存储结构介绍
- Linux系统的常见目录介绍
- Linux绝对路径和相对路径
    - 绝对路径与相对路径
    - 绝对路径：由根目录（/）开始写起的文件名或目录名称，是一个完整的路径
    - 相对路径：相对于目前路径的文件名写法（用于指代文件或文件夹路径）
        - ．代表当前的目录或者可以使用./来表示
        - ．．代表上—层目录或者也可以用../来代表
