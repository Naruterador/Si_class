## Linux系列课程说明
- 本教程所有的文档全部采用markdown格式
- 为什么不用word和PPT呢？
  - 根据调研，当前不管是软件公司还是互联网大厂，在做技术文档时基本都抛弃了word，首先是因为word太重了，其次不管是github仓库还是gitee都不支持word格式的在线预览，为了跟上时代脚步，所以决定弃用word
  - 至于为什么不用PPT，大家都是来学技术的，不是来看产品发布会的，PPT做起来既浪费时间又没啥屁用，上课直接用markdown的文档就行了。
- 教材中所使用的Linux发行版可以根据实际上课需求调整
- 由于技术的日新月异，该文档也会做一个长期更新
- 后续会添加更多的配置
