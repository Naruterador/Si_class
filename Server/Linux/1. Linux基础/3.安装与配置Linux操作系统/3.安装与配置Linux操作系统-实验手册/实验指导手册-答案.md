### 三、实验指导

#### 3.1、实验原理

- Linux系统提供了多达4种安装方式支持：CD-ROM/DVD启动安装、从硬盘安装、从NFS服务器安装、从FTP/HTTP服务器安装等。

#### 3.2、实验步骤
- 单击“创建新的虚拟机”选项，并在弹出的“新建虚拟机向导”界面中选择“自定义高级”单选按钮，然后单击“下一步”按钮
![[lab31.png]]
- 直接使用默认兼容性，选择“下一步”按钮
![[lab32.png]]
- 在这个的界面中，将客户机操作系统的类型选择为“Linux”，版本为“Centos 64位”，然后单击“下一步”按钮
![[lab34.png]]
- 取消“检查时启动”和“加入VMware”，选择稍后安装操作系统,然后单击“下一步”
![[lab33.png]]
- 这里可以自定义虚拟机名称，并在选择安装位置之后单击“下一步”按钮
![[lab35.png]]
- 选择CPU数量和处理器内核数量，这里我们选择默认，直接单击“下一步”按钮
![[lab36.png]]
- 在这个的界面中，建议将虚拟机系统内存的可用量设置为2GB，最低不应低于1GB
![[lab37.png]]
- 将虚拟机系统的网卡模式设置为桥接让虚拟机可以与宿主机相互通信，然后单击“下一步”按钮
![[lab38.png]]
- 输入和输出设备控制器使用默认配置，单击“下一步”按钮
![[lab39.png]]
- 硬盘接口类型也使用默认的SCSI(s)，单击“下一步”按钮
![[lab40.png]]
- 选择创建新硬盘，然后单击单击“下一步”按钮，将“最大磁盘大小”设置为50.0GB（默认即可），然后单击“下一步”按钮
![[lab41.png]]
- 硬盘文件命名使用默认名称，单击“下一步”按钮

- 双击光驱，选择ISO,单击“浏览”，选择Centos 8.3的镜像文件，将文件插入光驱：

- 在虚拟机管理界面中单击“开启此虚拟机”按钮后数秒就看到Centos8.3系统安装界面。在界面中，“Test this media & install CentOS Linux8”和“Troubleshooting”的作用分别是校验光盘完整性后再安装以及启动救援模式。此时通过键盘的方向键选择“Install CentOS Linux8”选项来直接安装系统
![[lab47.png]]
- 按回车键后开始加载安装镜像，所需时间在30秒-60秒，请耐心等待，选择系统的安装语言（简体中文）后单击“继续”按钮
![[lab48.png]]
- 在安装界面中单击“软件选择”选项
    - Centos系统的软件定制界面可以根据用户的需求来调整系统的基本环境，例如把Linux系统用作基础服务器、文件服务器、Web服务器或工作站等。此时只需在界面中单击选中“带GUI的服务器”单选按钮，然后单击左上角的“完成”按钮即可
![[lab49.png]]
- 返回到Centos系统安装主界面，单击“网络和主机名”选项后，单击右下角的配置按钮，然后选择IPv4设置，在方法选项中，可以将地址设置为手动或者自动，最后单击保存。单击右上角的开启按钮启动网卡，然后单击左上角的“完成”按钮
![[lab50-1.png]]
![[lab50-2.png]]
![[lab50-3.png]]

- 返回到Centos系统安装主界面，单击“安装目的地”选项后，在存储配置中单击“自定义”按钮，然后单击左上角的“完成”按钮
![[lab51-1.png]]
![[lab51-2.png]]

- 创建boot分区。在“新挂载点将使用以下分区方案”选中“标准分区”。单击“+”按钮，选择挂载点为“/boot”（也可以直接输入挂载点），容量大小设置为500MB，然后单击“添加挂载点”按钮。设置文件系统类型为“ext4”，剩下的分区按照以上方法一次建立即可。(注意：如果是EFI启动模式，还需要额外创建一个/boot/efi分区！)

![[lab52-3.png]]
- 分区都创建好后，单击左上角的“完成”按钮，，最后单击“接受更改”按钮完成分区
![[lab53.png]]

![[lab54.png]]
